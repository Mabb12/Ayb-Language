О Айбийском языке

Айбийский язык был создан ещё очень давно, но только сейчас был переделан. В этом языке минимальное количество слов для легкости, но сам он сложен в разговоре.

Глава Айбии и студии Манл является Мабб. Так же Мабб сделал этот сайт.

Примеры предложений

Простое предложение

Айбийский: Ама гуд су.

Русский: Я хороший человек.

Отрицание

Айбийский: Ама нот деш пыш.

Русский: Я не делаю предмет.

Составное слово

Айбийский: Нотёгудёпок кош.

Русский: Здесь алкоголь (букв. "не-хорошая-вода").

Вопрос

Айбийский: Ама гуд суда?

Русский: Я хороший человек?

Множественное число

Айбийский: Су-су деш фудёпри.

Русский: Люди делают салат (букв. еда-из-растения).

Правила Айбийского языка

1. Составные слова

Для создания новых понятий используйте разделитель "ё":

  • Нотёгудёпок → "алкоголь" (букв. "не-хорошая-вода")
  • Кошёсу → "дом" (букв. "человеческое-место")

2. Вопросы

Для образования вопроса добавьте частицу "да" в конец предложения:

  • Ама гуд су. → "Я хороший человек."
  • Ама гуд суда? → "Я хороший человек?"

3. Множественное число

Образуется путем удвоения корня через дефис:

  • Су → "человек"
  • Су-су → "люди"
  • При-при → "растения"

4. Порядок слов

Базовый порядок: Подлежащее + Сказуемое + Дополнение

  • Ама деш пыш. → "Я делаю предмет."
